# 国内优先权威资料采集说明

本目录保存“准备进入儿科 RAG 的权威来源清单”，目标不是一次性抓全网，而是建立可审计、可持续维护的知识源台账。

## 当前分层

- `auto`
  - 已知存在稳定公开直链，可由脚本自动下载
- `manual_review_required`
  - 官方公开来源，但具体 PDF/页面需要人工确认
- `licensed_manual_only`
  - 教材、会员内容、版权敏感资料，只能授权后人工导入

## 国内优先建议顺序

1. 国家卫生健康委员会
2. 中国疾病预防控制中心
3. 中华医学会儿科学分会
4. 中国医师协会儿科医师分会
5. 已授权教材

## 第一批建议锁定文档

建议先补这几类，足够支撑“发热/咳嗽/腹泻/过敏/生长发育/疫苗”六大核心场景：

- `0～6岁儿童健康管理服务规范`
- `儿童健康检查服务技术规范`
- `国家免疫规划疫苗儿童免疫程序及说明`
- `预防接种工作规范`
- 儿童发热相关指南/专家共识
- 儿童咳嗽相关指南/专家共识
- 儿童腹泻病诊疗指南/专家共识
- 儿童食物过敏相关指南/专家共识

## 落地流程

1. 在 `pediatric_authority_sources.json` 中找到对应来源
2. 确认资料来自官方或明确授权来源
3. 若存在稳定公开下载地址，补 `directUrl`
4. 运行 `python services/ai-engine/scripts/knowledge_cli.py fetch --source <id>`
5. 运行 `python services/ai-engine/scripts/knowledge_cli.py import`

## 采集底线

- 不使用非官方镜像替代官方来源
- 不抓取受版权限制但未获授权的教材/期刊全文
- 每份入库资料都必须可追溯到机构、原始链接、下载时间、本地哈希
